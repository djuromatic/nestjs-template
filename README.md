## Description

## Installation

```bash
# install dependencies
$ yarn app:install

# generate certificate
# you need to have mkcert installed
# https://formulae.brew.sh/formula/mkcert
$ yarn bizzllet:certificate

```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
